📊 Gestor de Ficheiros Excel
Exportar, descarregar e fazer backup de todos os dados em formato Excel ou JSON.
ℹ️
Como funciona
  • Exportar BD para Excel — preenche o template original com os dados actuais da aplicação e descarrega.
  • Download de Template — descarrega o ficheiro Excel original do servidor (sem dados da app).
  • Exportar Módulo — cria um Excel novo com os dados de um módulo específico.
  • Backup JSON — cópia de segurança em formato JSON para restauro rápido.
